# 💪 트레이너 데이터 관리 대시보드

> 트레이너가 직접 데이터를 추가하고 실시간으로 AI 챗봇 품질을 확인할 수 있는 웹 대시보드

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://your-app.streamlit.app)

## 📖 개요

이 대시보드는 피트니스 트레이너가 AI 챗봇에 데이터를 추가하고, 실시간으로 챗봇 품질(점수/티어)을 확인할 수 있는 웹 애플리케이션입니다.

### 주요 기능

- ✅ 실시간 데이터 현황 및 점수 확인
- ✅ 간편한 데이터 추가 폼
- ✅ 티어 시스템 (Bronze/Silver/Gold)
- ✅ 시각화 (게이지 차트, 막대 차트)
- ✅ 우선순위 기반 데이터 가이드

## 🚀 빠른 시작 (로컬 개발)

### 1. 저장소 클론

```bash
git clone https://github.com/YOUR_USERNAME/trainer-dashboard.git
cd trainer-dashboard
```

### 2. 가상환경 생성 및 활성화

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
```

### 3. 패키지 설치

```bash
pip install -r requirements.txt
```

### 4. 환경 변수 설정

```bash
cp env.example .env
# .env 파일을 열고 실제 DB 정보 입력
```

### 5. 대시보드 실행

```bash
streamlit run app.py
```

브라우저에서 자동으로 `http://localhost:8501`이 열립니다!

## 🎯 사용 방법

### Step 1: 현황 확인

1. 대시보드 접속
2. 상단에서 현재 티어와 점수 확인
3. "현황" 탭에서 카테고리별 달성률 확인

### Step 2: 필요한 데이터 확인

1. "필요한 데이터 (우선순위)" 섹션 확인
2. 🔴 긴급 카테고리부터 작성
3. 가이드 참고

### Step 3: 데이터 추가

1. "데이터 추가" 탭 클릭
2. 카테고리 선택
3. 폼 작성
4. "저장하기" 버튼 클릭
5. 점수 변화 확인!

### Step 4: 티어 달성

- 60점 달성 → 🥉 Bronze → 챗봇 활성화!
- 75점 달성 → 🥈 Silver → 권장 수준
- 85점 달성 → 🥇 Gold → 우수 수준

## 📋 FAQ

### Q: 데이터를 추가했는데 점수가 안 올라요

A: "데이터 새로고침" 버튼을 클릭하세요.

### Q: 티어가 언제 상승하나요?

A:

- Bronze: 60점 이상
- Silver: 75점 이상
- Gold: 85점 이상

### Q: 어떤 카테고리부터 작성하나요?

A: "필요한 데이터 (우선순위)" 섹션의 🔴 긴급 카테고리부터!

### Q: 데이터를 수정하거나 삭제할 수 있나요?

A: 현재 버전에서는 추가만 가능합니다.
수정/삭제는 DB에서 직접 하거나, 다음 버전에서 추가 예정입니다.

### Q: 여러 트레이너가 동시에 사용할 수 있나요?

A: 가능합니다! Trainer ID만 변경하면 됩니다.
